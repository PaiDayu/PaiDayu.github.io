搭建第一个hugo博客的过程

Apr 17, 2021
1 min read

下载hugo

1.打开官网下载列表[hugo下载地址] :https://github.com/gohugoio/hugo/releases

2.下载适配自己电脑系统的文件,我选的是hugo_0.82.0_Windows-64bit.zip

3.放到d盘,将其更改名字为hugo

4.右键我的电脑，点击属性–高级系统设置–环境变量–双击Path–新建–输入：D:\hugo

5.win+r打开运行，输入cmd打开系统自带的命令行工具，输入hugo version，如果出现以下提示表示安装成功：

6.hugo new site myblog表示会在当前根目录下创造一个myblog文件夹,里面就是网站结构,如图所示

下载一个hugo主题

1.[hugo主题下载地址]:https://themes.gohugo.io/

2.进入项目根目录下: cd myblog

sdd

3.输入git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c 命令将主题下载到项目根目录下

4.输入hugo server -t m10c –buildDrafts用来开启服务,这时我们的博客网站就已经启动了
